iSH + Alpine Linux on iPhone
Quick command reference & ideas to power-up your device

Heads-up: iSH emulates a Linux userspace (BusyBox + Alpine) in a sandbox. It’s awesome for CLI tools, scripting, editing, git, and learning — but it can’t access iOS hardware directly or run heavy background daemons.

Core commands (BusyBox/Alpine)

Command What it does
uname -a Kernel/OS info
whoami Current user
pwd Print working directory
ls -lAh List files (long/all/human)
cd Change directory
cp, mv, rm, mkdir, rmdir Copy, move/rename, remove, make/remove dirs
cat, head, tail, less View files
find . -name "foo*" Search files by name
grep -R "text" . Search inside files
du -sh * Show folder sizes
df -h Disk usage summary
ps, top, kill <pid> Process list / kill task
wget <url>, curl -LO <url> Download files
ping <host> Network reachability
chmod 755 file Change permissions

Package management with apk

Command Purpose
apk update Refresh package list
apk upgrade Upgrade installed packages
apk search <pkg> Find packages
apk add <pkg> Install a package
apk del <pkg> Remove a package

Creating your own user

apk add shadow
useradd -m -s /bin/ash <yourname>
passwd <yourname>
su - <yourname>

Starter-pack installs

apk add nano vim git curl wget zip unzip tar less
apk add python3 py3-pip py3-venv
apk add nodejs npm
apk add build-base cmake pkgconf
apk add openssh-client rsync openssl bind-tools net-tools
apk add ripgrep fd fzf bat tree ncdu tmux

What you can do with Alpine on iPhone

  • Run Python or Node.js scripts.
  • Clone, edit, and push GitHub projects.
  • Call APIs with curl or httpie.
  • Learn shell scripting & Linux basics.
  • Build small CLI tools with build-base.
  • SSH into servers & manage files remotely.

Prompt customization

# ~/.profile
export PS1="\\u@\\h:\\w\\$ "

Files created in iSH appear under Files → On My iPhone → iSH.
